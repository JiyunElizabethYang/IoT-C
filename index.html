Welcome to "Weather Tracker IoT"

Project Overview

Weather Tracker IoT is a software-based IoT project that simulates an ESP32 system using VS Code and Wokwi.

- The system retrieves real-time weather data from the Korea Meteorological Administration (KMA) through its API Hub and uses simulated GPS coordinates to provide location-based weather information.

- Based on the retrieved data, the system generates virtual sensor outputs and visual indicators, including LED signals and graphical visualizations on an OLED display, representing weather conditions such as temperature, humidity, rainfall, and wind.

- All processed information is presented in real time, demonstrating the integration of external data sources, location awareness, and sensor visualization within a fully virtual IoT framework.

Method

GPS Module
  • Runs a lightweight web server
  • Obtains the device’s internet IP address
⬇
IP-Based Location Detection
  • Converts the IP address into an approximate geographic location
  • Sets latitude and longitude coordinates
⬇
KMA API Hub
  • Retrieves real-time weather data in XML format
⬇
Weather Data Parsing Module
  • Temperature
  • Humidity
  • Rainfall
  • Wind speed
  • Current time
⬇
OLED Display #1 (Text)
  • Location information
  • Weather conditions
  • Current time
OLED Display #2 (Graphs)
  • Temperature graph
  • Humidity graph
  • Rainfall graph
  • Wind speed graph
⬇
Virtual LED Indicators
  • Red: Sunny
  • Blue: Rainy
  • Yellow: Windy

Benefits

Benefits of the Weather Tracker IoT Project

1. Real-Time Weather Monitoring
The system enables real-time tracking of weather conditions by retrieving live data from the Korea Meteorological Administration (KMA) API. This allows users to access up-to-date information such as temperature, humidity, rainfall, wind speed, and time.

2. Location-Based Weather Visualization
By simulating GPS coordinates and converting them into city and district names, the project provides location-specific weather information. This makes the system more realistic and demonstrates how IoT services can adapt data based on geographic location.

3. Effective Data Visualization
Weather data is presented using LED indicators and OLED graphs, making complex information easy to understand at a glance. For example, different LED colors represent different weather conditions, and graphs visually show changes in temperature, humidity, rainfall, and wind speed.
